#Горячие клавиши
#### Копировать:
CTRL+C
#### Вставить:
CTRL+V
#### Сохранить:
CTRL+S
#### Вырезать:
CTRL+X
#### Отмена:
CTRL+Z

# Работа с git
### Скачать проект:
```bash
git clone <URL>
```
### Учесть все файлы:
```bash
git add .
```
### Сохранить оммит:
```bash
git commit -m "..."
```
### Отправка в облако:
```bash
git push
```
